# 🛰️ AI Radar - AI 热点雷达

> 多源扫描 + 原文验证，每天十分钟掌握 AI 圈发生了什么。

## ✨ 功能

- 🔍 **六大平台并行搜索** — Twitter/X、Product Hunt、Reddit、Hacker News、arXiv、官方博客一站式采集
- ✅ **原文抓取验证** — 每条热点必须经过 `web_fetch` 抓取原文，拒绝搜索摘要凑数
- 📊 **五维打分筛选** — 领域匹配 / 实操价值 / 时效性 / 来源权威 / 热度信号，只留高分内容
- 📝 **结构化日报输出** — 按「博主实践 / 新产品 / 研究动态 / 厂商更新 / 社区热议」五大板块整理
- 🎯 **七大聚焦领域** — Vibe Coding、Claude 生态、AI Agent、AI 知识管理、模型更新、AI 新产品、海外热点

## 📦 安装

**方式一：vskills 命令**
```bash
vskills install ai-radar
```

**方式二：git clone**
```bash
git clone https://github.com/DoraRabbitYan/ai-radar.git
```

## 🚀 快速开始

对助手说：

```
开始今日选题
```

或：

```
采集AI热点
```

助手会自动执行两阶段采集：搜索发现 → 打分筛选 → 原文抓取 → 输出日报。

## 🧠 核心机制：两阶段采集

| 阶段 | 动作 | 说明 |
|------|------|------|
| **阶段一** | `web_search` 并行搜索 | 6 路并发搜索各平台，初筛发现 |
| **筛选** | 五维打分（0-10 分） | 领域匹配 + 实操价值 + 时效性 + 来源权威 + 热度信号 |
| **阶段二** | `web_fetch` 抓取原文 | 高分条目逐条抓取原文，基于实际内容提炼 |

> ⚠️ 核心原则：不得仅凭搜索摘要输出。每条热点必须经过原文验证。
